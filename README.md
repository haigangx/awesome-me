# 配置备份

备份方法请查看[这篇文章]()


## 说明

- 本配置仅在第一次从github上pull，第一次pull完成后需要再库目录运行`re_build.sh`脚本来构建远程库和本地配置文件的联系，以后每次配置改变只需要`add,commit,push`即可

- autorun_ubuntu.conf
  新ubuntu开发环境搭建

- init.vim nvim配置，位于~/.config/nvim/

- .tmux.conf tmux配置，位于~/

## Note

不要在当前目录下手动修改`autorun_ubuntu.conf`,`init.vim`,`.tmux.conf`等配置文件
